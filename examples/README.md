# Examples

Run these from the repository root with Bun.

## `scaled-basic.ts`

Demonstrates the Stage 3 `@Scaled()` decorator on a class method.

```bash
bun run examples/scaled-basic.ts
```

## `worker-class-basic.ts`

Demonstrates the class-level `@WorkerClass()` decorator with a shared worker pool.

```bash
bun run examples/worker-class-basic.ts
```
